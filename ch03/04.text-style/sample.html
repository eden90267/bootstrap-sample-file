CSS3 互動網頁開發超好玩

課程介紹

手機網頁已成為企業需求，你是否認為學會 RWD 就完整了？行動裝置對 CSS3 的支援度比起電腦來說要更加的完整全面，本課程特別針對 CSS3 的各項新增屬性、應用方式、應用問題、詳細的屬性實作…等有所安排外，更讓您學習到 google 上查不到的 CSS3 應用細節與實作會遇到的問題，在手機網頁設計上面實用度完全爆表．

在此課程中，老師針對教學設計了原創的教學範例、酷炫實例、業界常見效果及應用、專案應用範例…等多種層面的實務範例，更重要的是所有學習到的範例都可以延伸更多的實務應用，加強你的 CSS 結構觀念、應用觀念、選取器的選取效能改善、CSS3 與 jQuery 的合併應用與使用時機．

許多設計師都說既然主流瀏覽器不支援，學習 CSS3 要做什麼？目前的 CSS3 雖然在某萬惡的瀏覽器上面支援度有稍稍的不完整，但在 Chrome 以及 Firefox 上的支援度，已經是非常完整了，而在行動裝置上 CSS3 的支援度已超過9成以上都支援呢！面對著未來行動上網越加盛行的時代，越是完整掌握 CSS3 的技術，才越能掌控先機，並快速開發你的行動裝置與桌上型電腦網站。

對於許多人關注的 HTML5 JavaScript 動畫技術來說，CSS3 的動畫效能對行動裝置具有更佳的優勢，當別人還在學習用程式撰寫動畫時，你已經可以利用簡單的 CSS3 就完成了，此外，針對於視覺效果越來越豐富的網頁設計趨勢，CSS3 更是你不可不學習的一門技術。

Dreamweaver 近幾年不斷的更新版本，但事實上對於 CSS 的功能卻不見有很大的進步，許多入門使用者往往只會利用 Dreamweaver 內建的 CSS 編輯器來製作網頁，然 CSS 本身有許多屬性是在 Dreamweaver CSS 編輯器中無法看到的，更不用談新的 CSS3 屬性了！學習新的 CSS3 語法，讓你不用受限於軟體本身的限制，想怎麼發揮創意就怎麼發揮！

Bootstrap3 RWD網頁開發技快速上手

課程介紹

讓您學會即使不熟 CSS 也能快速使用 Bootstrap 完成 RWD 網頁設計的技能．課程內容經過詳實的規劃及安排，讓您輕鬆學習，快速掌控！並在教學中讓您學會 mobile first 網頁設計，不管是網頁視覺設計師還是前端工程師，具備 RWD 網頁的製作能力！課程中使用會 Sublime Text 開發工具的操作．讓您網頁設計技能立馬升級！編輯速度加倍提升！

在這個行動裝置盛行的時代，不管是網頁視覺設計師還是前端工程師，具備 RWD 網頁設計的製作能力已是一項必備技能，RWD 網頁已經是製作網站的標準要求，以往全部手刻 CSS RWD 的網頁曠日費時，老闆跟客戶在不瞭解 RWD 網頁的情況下更是抱怨工作速度太過緩慢，現在我們可以利用 Bootstrap 快速開發建置RWD 網站，訊速完成一個 RWD 網頁絕對不是夢，本課程將教您如何利用 Bootstrap 的各項編排系統及組件，快速完成您的 RWD 網站，並保留網頁版面修改及擴充的彈性。
